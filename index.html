PacMan

Did you know that the original title of “Pac-Man” was “Puck Man”?? Unlike many 40-year-old video games, this nostalgic classic is still available on many gaming platforms. So you can imagine my nerves and excitement, when I was challenged to create my own mini Pac-Man game! This project allowed me to use JavaScript to scale by creating multiple “Pac-Men”, which move in dynamic motion across the screen. I hope you enjoy this little game as much as I do!
